    Install Guide #

    One-line binary install #

    Install the correct binary for your operating system and architecture in ./bin with a single command:

    $ sh -c "$(curl -fsLS chezmoi.io/get)"

    Or, if you have wget instead of curl:

    $ sh -c "$(wget -qO- chezmoi.io/get)"

    If you already have a dotfiles repo using chezmoi on GitHub at https://github.com/<github-username>/dotfiles then you can install chezmoi and your dotfiles with the single command:

    $ sh -c "$(curl -fsLS chezmoi.io/get)" -- init --apply <github-username>

    On systems with Powershell, use:

    (iwr -UseBasicParsing https://chezmoi.io/get.ps1).Content | powershell -c -

    From source #

    Download, build, and install chezmoi for your system:

    $ git clone https://github.com/twpayne/chezmoi.git
$ cd chezmoi
$ make install

    Building chezmoi requires Go 1.16 or later.
